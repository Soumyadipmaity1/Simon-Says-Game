Simon Says

Press any key / Tap to start the game!

Simon Game playing Guide

  • Press any key / Tap to start the game but do not tap any of the boxes.
  • When a color Box flash with white color remember the box and click / tap on it.
  • If you click / tap on the correct box then a new / same box flash again.
  • Every correct click/tap change the color of the box to green.
  • Now you have to click / tap all the previous boxes and then this box.
  • In the sequence if you click / tap on any wrong box then that will be wrong answer.
  • If your answer is wrong then background will be flash with red color and 'Game Over' then press any key / tap to restart again.
